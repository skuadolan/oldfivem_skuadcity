Selamat datang di Sekolah Mengemudi


Semua warga kota Los Santos harus lulus ujian sebelum dapat mengemudi.
Luangkan waktu Anda, jawablah dengan akal sehat, dan jangan jawab secara acak.

Tes Teori
- Tes Teori berharga $ 200, ini tidak akan dikembalikan jika Anda gagal tes.
- Jangan takut, sekolah mengemudi menerima kredit, tapi hati-hati jangan sampai terlilit hutang.
- Jika Anda gagal dalam tes pertama kali, Anda tidak dapat segera mengambilnya kembali, Anda harus mengambilnya nanti tanggal.

Tes mengemudi
- Tes Mengemudi berharga $ 500, sama seperti tes teori, pembayaran ini tidak akan dikembalikan jika Anda gagal.
- Pastikan Anda tetap waspada saat mengemudi, dan menghindari kecelakaan!
